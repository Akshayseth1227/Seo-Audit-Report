# Seo-Audit-Report
Website: https://www.advancells.com/
DA: 34 & Spam Score: 1%
Prepared by: Akshay Seth
Email ID: akshayseth.79@gmail.com

This audit is limited to key pages and site-wide observations to demonstrate SEO skills.

On Page SEO
Meta title and description are Missing or not optimized on some pages.
Keyword optimization is not good.
Some pages are not linked properly.
Most of the images have no alt atributes.

Recommendations

Write unique meta titles & descriptions for each page
Use one clear H1 per page with main keyword
Add internal links between treatment pages
Optimize images with descriptive alt text

OFF Page SEO
Limited backlink profile
Low presence on authoritative medical websites

Recommendations

Build backlinks from blog post, guest posting, social bookmarking, press release and directories

Technical SEO
Large image size
Page speed is slow
Crawling is not good
Some elements may not be fully optimized for mobile users

Recommendations

Reduce image size
Minimize css, javascript and image size for website speed
Mobile responsiveness

Content
The website content is informative but not fully optimized for search intent.
There is limited use of targeted keywords, FAQs, and blog content that users actively search for.
